##Solarized Colorscheme for PHPStorm 6

###Installation

####Mac OSX
PHPStorm 6: ~/Library/Preferences/WebIde60/colors

####Ubuntu
PHPStorm 6: ~/.WebIde60/config/colors (Is this still correct?)

Font:
Monaco 12pt is nice
